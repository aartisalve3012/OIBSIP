# ATM Interface

## Overview

This is a Java-based ATM Interface developed as part of the Oasis Infobyte Java Development Internship.

The project simulates the basic functionalities of an Automated Teller Machine (ATM), allowing users to perform banking operations such as checking account balance, depositing money, withdrawing money, and exiting the system.

## Features

* Check Account Balance
* Deposit Money
* Withdraw Money
* Menu-Driven Interface
* Input Validation
* User-Friendly Console Interaction

## Technologies Used

* Java
* VS Code
* GitHub

## How to Run

### Compile the Program

```bash
javac ATMInterface.java
```

### Run the Program

```bash
java ATMInterface
```

## Functionalities

### Check Balance

Displays the current account balance.

### Deposit Money

Allows the user to add money to the account.

### Withdraw Money

Allows the user to withdraw money if sufficient balance is available.

### Exit

Terminates the ATM session safely.
