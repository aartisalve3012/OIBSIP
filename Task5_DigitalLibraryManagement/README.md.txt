# Digital Library Management System

## Overview

Digital Library Management System is a Java-based application developed using JDBC and MySQL. The system automates library operations such as managing books, users, issuing books, and returning books.

## Technologies Used

* Java
* JDBC
* MySQL
* XAMPP
* VS Code

## Features

* Add Book
* View Books
* Search Book
* Update Book
* Delete Book
* Add User
* View Users
* Delete User
* Issue Book
* Return Book
* View Issued Books

## Database Tables

* admin
* books
* users
* issued_books

## How to Run

1. Start MySQL from XAMPP.
2. Import the library.sql database.
3. Add MySQL Connector JAR to the project.
4. Run Main.java.
